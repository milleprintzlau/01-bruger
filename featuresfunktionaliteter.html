Features/funktionaliteter:

Like billeder + kommentare:

  • dobbelt tap + tryk på hjerte og tryk på hjertet ved kommenrtar
Vidersende kommunikere:
  • Du kan sende til kontakter, grupper og fremmede
Afvisning af kommunikation/accept:
  • En uønskede besked effekt kommer frem, som man så kan trykke ja eller nej til.
Livestream:
  • Man kan tilføje folk
  • Sende dit livestream
  • Live feed count
  • Man kan kommentere på skrift og ved tale - det kan også slåes fra.
  • Man kan vinke, man kan være flere på live stream sammen
Mystory:
  • Sende en direkte besked for mystoryen
  • videresende mystoryen til kontakter
  • anmelde en mystory
  • man kan se hvor mange der ser ens mystory
  • man kan ligge tekst på sin mystory
  • man kan sætte location på (gør den oftest automatisk pga. Location track)
  • En mystory kan både være videoer og billeder
  • du kan lave effekter/redigere dine billeder
Profil:
  • skive en bio
  • ligge link op
  • oversætte tekst
  • se ens egne billeder - se ens farvorit billeder - se tagged billeder
  • se hvem af ens venner som følger denne profil
  • man kan sende beskeder
  • man kan se hvor mange der følger og hvor mange der følges
  • hvor mange opslag hun har
  • forslående kontakter
  • holde et billede ind og zoom
  • invitere venner til IG
  • gemme opslag til senere brug
  • man kan unfollow
Kontakter:
  • man kan connecte til sine telefoner kontakter
  • man kan slå sin facebook venner til
  • man kan søge på brugernavn etc.
sælge/annoncere:
  • man kan oprette en virksomhedsprofil
  • man kan få inblik i viksomheden
  • ved betaling kan man promovere, sammenkom til facebook side
  • kan man sælge direkte, man kan swipe for link, man kan komme indirekte ind på sælgerens hjemmeside
  • gennem IG app
